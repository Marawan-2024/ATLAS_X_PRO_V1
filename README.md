# ATLAS_X_PRO_V1
ATLAS-X Pro Antivirus is a modern, production-ready antivirus solution built for Windows with a professional PyQt5 GUI, real-time file system monitoring, and a modular multi-layered engine. It combines signature-based scanning, heuristic analysis, and an AI placeholder scoring system to detect potential threats with transparency and reliability.

Key Features:
- Professional GUI with Dark Mode for enhanced user experience
- Real-time File System Watchdog monitoring all file creation events
- Live Charts Dashboard to visualize scanning activity and performance
- Modular engine: Signature Scanner, Heuristic Analysis, AI Risk Scoring
- File Quarantine system (placeholder for future expansion)
- Windows-ready structure with easy deployment
- Lightweight and extendable for future AI or malware detection improvements
- Clear and explainable risk scores for user transparency
- Fully documented with step-by-step deployment instructions

This repository is designed for educational and professional purposes. ATLAS-X Pro is structured to be easily converted into an EXE or installer for distribution and can serve as a foundation for advanced antivirus development or security software experimentation.

Installation:
1. Install Python 3.10 or higher
2. pip install -r requirements.txt
3. python main.py

Contributions, suggestions, and security enhancements are welcome. ATLAS-X Pro aims to provide a robust, user-friendly antivirus experience while remaining extendable for advanced security development.
